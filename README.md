XGalaga JS
==========

Description
-----------

This is a JavaScript port of the famous arcade game
[XGalaga](http://rumsey.org/xgal.html) from Joe Rumsey.

You can play it right here:

[PLAY XGALAGA JS!](http://kayahr.github.com/xgalaga-js/)
